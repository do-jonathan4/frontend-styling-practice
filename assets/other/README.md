## Instructions
### Command Line Input
```
node index.js [[operator][number][time range]] [number of commits]
```
### Parameters
`operator` <br/>
`+` for time forward <br/>
`-` for time backwards <br/>

`number` <br/>
amount of months of years <br/>

`time range` <br/>
`M` or `m` for month <br/>
`y` for year <br/>

### Example
```node index.js +2M 50```
- make `50` commits `2` `months` into the `future`
